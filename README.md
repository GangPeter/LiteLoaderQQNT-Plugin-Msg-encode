# LiteLoaderQQNT-Plugin-Msg-encode
基于LiteLoaderQQNT的消息加密解密插件，适用于朋友间娱乐
# 功能介绍
- 自定义密码
- 自定义显示模式
- 添加加密发送按钮
- 右键菜单添加"加密+1"
- 右键菜单添加"解密"
- 自动解密最新消息
